User prefers detailed code comments with line-by-line explanations of what each line does.

**Why:** User wants to be able to review and understand the code later.

**How to apply:** Add comments to every line explaining what it does. For non-intuitive operations, always include a comment explaining the reason/why this approach was chosen.
