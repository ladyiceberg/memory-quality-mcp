User prefers receiving answers in "结论先、解释后" (conclusion first, explanation second) order. This is their preferred reading style.

**Why:** The user has explicitly stated this is how they like to process information - they find it more efficient to know the conclusion first before diving into details.

**How to apply:** Always structure responses with the conclusion/answer at the beginning, followed by supporting explanation and details.
